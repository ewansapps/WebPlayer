# WebPlayer
*Play your music with elegance*

# Features
- Support MP3, Vorbis, FLAC, Opus and more
- Metadata reading
- Synced lyrics (local or LRCLIB)
- Playlists creation
- And more !
  
# Build
```bash
npm install
npm run electron:build
```

# Why was this player created ?
I would like to create the perfect player for me after tested many music player.
